# Multi-Agent Code Optimizer

This project implements a multi-agent AI-based smart code optimization tool. It uses a combination of multiple agents to perform tasks like code analysis, performance optimization, and code compliance checks. The agents collaborate to optimize code and ensure high-quality standards.

## Features:
- Code analysis by one agent
- Performance optimization by another agent
- Code compliance checks by a third agent
- Integration with GitHub Actions for automated optimization

## Getting Started:
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Use the optimizer by importing the `MultiAgentCodeOptimizer` class from `src.multi_agent_optimizer`.
